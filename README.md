# Meteobot modul

Meteobot modul for Drupal 7

## API

API link: [site_address]/api/meteo/[period].

Parametar [period] is optional.

| Parametar | API return   |
|:---------:|:------------:|
| if omited | daily data   |
| [daily]   | daily data   |
| [weekly]  | weekly data  |
| [monthly] | monthly data |
